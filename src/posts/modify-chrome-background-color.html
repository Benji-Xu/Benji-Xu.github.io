保护视力，修改 Chrome 浏览器背景颜色
2012-07-26 ｜ # 折腾
▼ 文章目录

周良一直在用Google的Chrome浏览器，对于我这个夜猫子来说，最大的痛苦莫过于大半夜使用Chrome浏览器，因为白色对眼睛伤害实在太大，必须要改点，换成浅绿色就最好了。觉得以前用 QQ 管家的护眼功能，就可以把网页上所有的白色变为浅绿色。不过我不太喜欢这样，很多设计精美的站点，都是因为腾讯的这一坑爹功能，而变得不堪入目。

所以，自己动手丰衣足食，直接从根本入手，研究一下 Chrome 浏览器的CSS，最终得到如下方法，可以实现修改Chrome浏览器空白页的背景颜色。

打开 Chrome 浏览器的安装目录，找到 custom.css 这个 CSS 文件并修改，加入这句话：

html, body {background-color: #CCE8CC!important;

其中 #CCE8CC 是颜色的码值。保存后浏览器的背景色会瞬间改变。

另外，如果你找不到 Chrome 浏览器的安装目录，你可以通过查看 Chrome 快捷键属性得到安装路径。

我的文件地址在：...\AppData\Local\Google\Chrome\User Data\Default\User StyleSheets 里面。

另外，如果日后你不需要修改 Chrome 浏览器背景颜色，你可以在周良提供的代码前面，加上加两斜杠 //，注释掉添加的代码，变为如下代码即可。

//html, body {background-color: #CCE8CC!important;}

做完上述步骤之后，你的 Chrome 浏览器背景就可以自定义了，想换什么颜色就换什么颜色！
