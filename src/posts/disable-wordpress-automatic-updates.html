WordPress 技巧：禁用 WordPress 自动更新
2014-03-15 ｜ # WordPress
▼ 文章目录
  1. 1. 方法一：通过 wp-config.php 屏蔽
  2. 2. 方法二：FTP 设置文件夹权限为不可写

随着 WordPress 的不断更新，官方早在3.7的版本中就添加了 WordPress 自动更新的功能，考虑到很多朋友在使用很早以前的 WordPress主题和插件，也许会造成后台的错误提示或者前台网页出错，所以周良这里分享一下禁用 WordPress 自动更新的几种方法。

方法一：通过 wp-config.php 屏蔽

把下面这段代码加到 wp-config.php 文件中即可禁用 WordPress 自动更新。

// 禁用 WordPress 自动更新
define( 'WP_AUTO_UPDATE_CORE', false );

方法二：FTP 设置文件夹权限为不可写

如果你打算永远不升级你的 WordPress，那么我建议你可以在你的服务器中把WordPress文件夹给设置「不可写」权限，这样不仅可以禁用自动更新，还可以防止 WordPress 系统文件被恶意 PHP 代码感染。（至于这么设置文件夹权限，我推荐大家 Google 一下）
