WordPress 技巧：如何禁用图片压缩功能
2012-04-07 ｜ # WordPress
▼ 文章目录

对于设计教程类、摄影类、各种图片分享类型的网站来说，图片的高清与否是很重要的。使用 WordPress 建站是很多人理想的选择，但是 WordPress 的在线图片上传压缩功能却让很多图片类型的站长感到头疼，于是不得不使用付费的图片存储空间。所以周良在这里分享一下这个WordPress 技巧

禁用WordPress的这项功能十分的简单，只需要在你使用的主题functions.php文件中添加下面的代码即可解决：

<?php
add_filter('jpeg_quality', function($arg){return 100;});
?>
