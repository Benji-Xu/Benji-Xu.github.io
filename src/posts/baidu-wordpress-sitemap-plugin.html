百度站长推出 WordPress 结构化数据插件，优化内容收录

  2. 2. WordPress 结构化数据插件安装和使用：

刚刚回到家，去百度站长平台逛了一圈，发现百度站长平台已经为 WordPress 站点推出了结构化数据插件。

什么是结构化数据工具那？ 简单介绍一下，结构化数据工具是百度引入优质结构化数据的入口，现在互联网中的资源类型越来越多，内容越来越丰富，为了给用户展示更丰富的搜索结果，同时给予网站内容更合适的展现，提高搜索结果的点击率，百度站长平台推出了结构化数据工具，同时为了方便 WordPress 站点用户使用，百度站长平台也推出了相应的 WordPress 结构化数据插件。

安装结构化数据插件，能又快又全的向百度提交论坛网页及内容。帮助百度 Spider 更好的了解您的网站，帮助收录，对于优质的数据资源，可应用结构化摘要样式优化网站在百度搜索结果页的展现。百度前期只为 Discuz！用户提供了这款插件，但是今天他为 WordPress 系统开发了插件，可见 WordPress 的重要程度。Discuz! 结构化数据插件和 WordPress 结构化数据插件，二者的功能也基本一致。

WordPress 结构化数据插件功能介绍：

  • 自动推送 Sitemap，优化内容收录。结构化数据插件会遍历论坛帖子页历史URL数据，可以帮你自动生成sitemap提交到百度。
  • 实时推送 WordPress 站点新内容，加快收录。有新的文章发布，立即将文章网址及内容推送至百度站长平台，站内页面发生变化，百度也能实时更新相应数据。
  • 优质内容，优化展现，提升用户点击率。应用结构化摘要优化展现，提升结果点击率，让你的 WordPress 站点流量节节攀升。

启用本插件后，我想你就可以抛弃掉一些专门为百度准备的 Sitemap 插件了，本插件是百度官方作品，效果当然会比直接去站长平台提交 Sitemap 站点地图效果好。

WordPress 结构化数据插件安装和使用：

  1. 通过本文下方链接下载文件
  2. 解压文件到 WordPress 的 wp-content/plugins 目录下
  3. 进入后台，启用插件，验证站点地址。
  4. 通过插件设置页面的「近期提交历史」即可查看百度结构化数据插件提交的数据
