新窗口打开 WordPress 文章中的所有链接
2015-02-07 ｜ # WordPress
▼ 文章目录

如果你希望 WordPress 文章中的所有链接都在新窗口中打开，只需在你的主题的 functions.php 文件中添加以下代码即可：

function autoblank($text) {
	$text = str_replace('<a', '<a target="_blank"', $text);
	return $text;
}
add_filter('the_content', 'autoblank');

将以上代码粘贴到 functions.php 文件的最后一个 ?> 之前，这样当访问者点击文章中的任何链接时，链接都会在新窗口中打开。这种方法可以提升用户体验，避免用户离开当前页面。
