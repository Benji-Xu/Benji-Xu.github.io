MP6 插件：带给你 WordPress 后台扁平化的体验
2013-06-28 ｜ # WordPress
▼ 文章目录

自从IOS7开始走扁平化路线之后，国内对扁平化这一理念可谓褒贬不一。而现在，就如本文篇标题所说的那样，WordPress 后台也可以实现扁平化了。或许你已经厌倦了当前的WordPress后台管理界面，那么我相信 MP6 这一款插件，一定可以给你带来全新的体验。

MP6 是由 WordPress 官方团队制作的一款插件，用来测试一些不适合在主线（Trunk）版本中上线的 idea。

安装最新的版本后，最直观的感觉就是整个后台界面扁平化了，没有了以前的渐变效果，所有的图标也都是通过 webfont 实现，并且整个后台采用了响应式设计，以后在手机上也能轻松使用 WordPress 后台。

扁平化的 WordPress 文章撰写页面


手机访问后台自适应显示

或许有一天，这款插件所带来的扁平化体验后台，会成为 WordPress 默认的管理后台，不过这也并不是肯定的。如果你有好的想法，可以到官方社区参与讨论，当然最终能否采用得由 matt（WordPress 创始人）来决定。

插件下载地址：http://wordpress.org/plugins/mp6/
