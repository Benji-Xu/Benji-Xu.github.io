cPanel 面板主机定时自动备份到异地 FTP
2011-12-18 ｜ # 折腾
▼ 文章目录

网站数据虽然不是很多，但出问题了的话也是很麻烦，所以从蓝冰那里找了这么一个备份脚本，许多网站给出的脚本都是错误的，根本就不能用，所以希望这个教程对您有用。

你可以点这里下载 CP全局备份.zip (原作者链接 )

该脚本只支持CP面板，不支持 DA 面板。
将该文件放到 /home/xxx/目录，其中 xxx 为你的用户名 该目录是不可以被访问的。记得修改其中的数据

如果你的 CP 用的 SSL 访问，就是 https 和 2083，记得修改以下

// 安全模式 $secure = 1; // 0为标准http, 1为ssl(需要主机支持ssl)

选择每天一次。 命令为  /usr/bin/php -q /home/xxx/fullbackup.php （其中xxx为你的用户名） 好了，每天凌晨你的邮箱都会收到通知，你可以到你的FTP查看备份信息.
