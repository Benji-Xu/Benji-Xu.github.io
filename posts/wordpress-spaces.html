WordPress 写作时，如何输入空格
2011-07-28 ｜ # WordPress
▼ 文章目录
  1. 1. 方法一
  2. 2. 方法二
  3. 3. 方法三

在使用 Wordpress 写文章时，很多朋友会遇到这样的困难：明明在文章每段的开头打上了空格，但最后发表出来的文章，几乎都是定格写的！今天周良教大家一个方法，可以在 WordPress 写文章时，添加空格，并且发表出来以后空格不会被和谐掉！

方法一

  1. 切换到HTML下，（可视化状态下本方法无效）；
  2. 打开中文输入法（ＱＱ拼音什么的，中文输入状态即可）；
  3. shift+空格，切换到全角（ＱＱ拼音下面的太阳图标）；
  4. 在想要缩进的段落前输入空格；
  5. 发布、更新就可以看到效果了（此时绝对不能再切换到可视化，否则会前功尽弃的）

方法二

在 WordPress 对应主题的 style.css 文件中找到 p {……}，向{……}中加入缩进代码text-indent:2em;，即形式为p {text-indent:2em;}这样就可以实现首行缩进了！（这样做的缺点是每次都缩进了，不想缩进的也缩进了）

但是不建议这种方法，因为当你发表文章的时候，只要是<p></p>之间的内容都空两格，比如 h 标签，这样就只要是一段的开头就空两格，这样就很麻烦。

方法三

就是在类似于 Ｗindows Live Writer 这样的本地电脑编辑器中发布博客。在WLW中发布，要实现首行缩进需要在「日志」菜单中点击「编辑日志设置」，然后选择「高级」，将「标记类型」改为「HTML」，并且保存。

WLW 还是很好用的，在 Windows 7 下安装很方便，不知道其他操作系统如何。但是仍然建议用全角模式下打空格，因为这样才是标准的中文排版。

PS：原本周良还为大家准备了一个方法，属于在 HTML 下添加颜色代码的方法，以前一直用，刚刚测试了一下，可以已经不能在 WordPress 3.2 下，所以不分享了。
