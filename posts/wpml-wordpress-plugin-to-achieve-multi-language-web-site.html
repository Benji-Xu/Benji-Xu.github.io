WPML 插件实现 WordPress 多语言网站
2011-10-29 ｜ # WordPress
▼ 文章目录
  1. 1. WPML插件特点：

今天给大家推荐一个插件：WPML，是一个最强大的 WordPress 多语言插件 ，可以让我们轻松地实现 WordPress 网站的多语言，而且插件的设置十分简单。

WPML插件特点：

  1. 支持三种方式区别不同的语言版本
  • 在不同目录中使用不同语言（简体中文、英语等）
  • 各语言使用不同的域
  • 语言名称添加为参数

  1. 实现插件和主题的本地化

  2. 可以实现评论翻译

  3. 提供专业的翻译（收费）

  4. 自由添加支持的语言

WPML 提供两种方式翻译内容：

  1. 手动，通过点击「+」图标
  2. 使用WPML的翻译管理模块

WPML插件下载：https://wpml.org/documentation/getting-started-guide
