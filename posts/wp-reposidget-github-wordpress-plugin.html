WP Reposidget：快速插入 Github 项目到 WordPress
2014-07-23 ｜ # WordPress
▼ 文章目录

Git 是一个分布式的版本控制系统，而 GitHub 可以托管各种代码库，并提供一个 Web 界面，但与其它像 SourceForge 或 Google Code 这样的服务不同，GitHub 的独特卖点在于从另外一个项目进行分支的简易性。为一个项目贡献代码非常简单：首先点击项目站点的 fork 的按钮，然后将代码检出并将修改加入到刚才分出的代码库中，最后通过内建的 pull request 机制向项目负责人申请代码合并。已经有人将 GitHub 称为代码玩家的 MySpace。

而 WordPress 作为世界上最大的开源博客平台，有许许多多的 Geek、Coder 在使用它构建自己的 Blog。为了方便插入 Github 项目到 WordPress 中，就诞生了 WP Reposidget 这款 WordPress 插件。

WP Reposidget 插件可以通过最简便的方法在 WordPress 文章/页面内插入 Github 的项目组件，让你的读者以最佳的方式看到你列举的 Github 项目。

具体插入到 WordPress 页面内的效果，其实就是 Github 的代码挂件，请看文章最上方截图，依靠一个短代码即可实现。

［repo path="Github 用户名/项目名称"]

WP Reposidget 插件下载地址：http://wordpress.org/plugins/wp-reposidget/
